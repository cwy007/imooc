# transform

transform: none | transform-function [transform-function];

CSS3的变形（Transform）属性让元素在一个坐标系统中变形。这个属性包含一系列变形函数，可以移动、旋转和缩放元素。

rotate()    旋转
translate() 平移
scale()     缩放
skew()      扭曲或斜切
