内阴影 可以有负值
/外阴影 从左上角开始

box-shadow: h-shadow v-shadow blur spread color inset;

box-shadow的值对比：
（1）水平阴影的位置。允许负值。必写；
（2）垂直阴影的位置。允许负值。必写；
（3）模糊距离。可选；
（4）阴影的尺寸。可选；
（5）阴影的颜色。请参阅 CSS 颜色值。可选；
（6）可选。将外部阴影 (outset) 改为内部阴影。
